# CarDealership
CS340 Project

### Instructions
1. Fork project
2. Open Intellij CE to welcome screen 
3. Select `Check out from Version Control`
4. Select `GitHub`
5. Select `CarDealership.git`
6. Select `Clone`
7. Make changes
8. Select `VCS`
9. Commit & Push
10. Go back to GitHub & select new pull request
